# tone9 fixtures

This directory contains the first regression fixtures extracted from the working chat.

## live/

Final or user-touched outline ODTs. These are visual/layout authorities, not source
text repositories. They should be used to test that generated output preserves the
approved compact two-page shape.

## source/

Input files used during the trial builds: rubrics HTML, liturgy HTML, and Minaion
ODT files. These are parser/extractor fixtures for future tone9 development.

## Current fixture set

- `2026-06-08-tone-ii-theodorou`: §1c six-stichira service, Tone II, Matins Gospel III.
- `2026-06-22-tone-iv-efsevios`: §1a simple service, Tone IV, Matins Gospel V.
- `2026-06-28-tone-iii-iona-polyeleos`: §1d Polyeleos service, Tone III, Matins Gospel IV.
